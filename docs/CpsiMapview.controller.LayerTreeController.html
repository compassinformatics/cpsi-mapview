(static) structureMode

Mode in order to steer how the layers in tree will be structured. At the moment 'BASELAYER_OVERLAY' will divide the layers in 2 folders 'Base Layers' and 'Overlays' (depending on their property 'isBaseLayer'). All other settings will result in a flat layer list.

(static) makeLayerStore()

This method assigns an instance of the GeoExt class `GeoExt.data.store.LayersTree` based on the connected OL #map to the view. The layers of the `ol.Map` are restructured and divided into groups based on the JSON tree structure loaded in #loadTreeStructure. This assures that the layers will appear in different folders in this TreePanel (as defined in the tree structure JSON).
