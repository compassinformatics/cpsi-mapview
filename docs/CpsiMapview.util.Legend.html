(static) createGetLegendGraphicUrl(layer) → {String}

Creates a WMS getLegendGraphic request URL for the given OL layer. For WMS it uses the standard WMS mechanism / information. For WFS the assumption is that MapServer uses the same configuration for WMS and WFS so we can create the getLegendGraphic the same way. STYLE parameter is derived by the corresponding SLD file name (see #getWmsStyleFromSldFile).
