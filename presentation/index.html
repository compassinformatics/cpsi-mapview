Compass Informatics MapView

Contributors

### Overview - Desktop GIS replacement - Built on open services such as WMS and WFS - Reusable components and plugins - Powerful grid component

JavaScript Libraries

### JavaScript Libraries - [OpenLayers](https://openlayers.org/) (v6.5) - [Proj4JS](https://github.com/proj4js/proj4js) (v2.7) - [GeoExt](https://geoext.github.io/geoext/) (v4.0) - [ExtJS](https://www.sencha.com/products/extjs/) (v6.7) - [BasiGX](https://github.com/terrestris/BasiGX) (v6.7) - [GeoStyler](https://geostyler.github.io/geostyler-demo/) for SLD and OpenLayers styles - [Font GIS](https://viglino.github.io/font-gis/) for icons See [package.json](https://github.com/compassinformatics/cpsi-mapview/blob/master/package.json) for details
